Implement greedy decoding (argmax) for language model inference. Given a 2D array of logits with shape \(B \times V\), where \(B\) is the batch size and \(V\) is the vocabulary size, select the token index with the highest logit value for each sequence in the batch. This is the simplest decoding strategy used in autoregressive language models.

Implementation Requirements

  • External libraries are not permitted
  • The solve function signature must remain unchanged
  • logits is a 2D array of 32-bit floats with shape (batch_size, vocab_size)
  • tokens is a 1D output array of 32-bit integers with shape (batch_size,)
  • For each row in logits, find the index of the maximum value and write it to tokens[i]
  • If multiple positions have the same maximum value, select the smallest index (first max)

Example 1:

\[ \text{logits} = \begin{bmatrix} 0.1 & 0.5 & 0.2 & 0.9 & 0.3 & 0.7 & 0.4 & 0.6 \\ 0.8 & 0.3 & 0.1 & 0.2 & 0.9 & 0.4 & 0.5 & 0.7 \end{bmatrix} \] \[ \text{tokens} = \begin{bmatrix} 3 & 4 \end{bmatrix} \]

Constraints

  • 1 ≤ batch_size ≤ 4,096
  • 1 ≤ vocab_size ≤ 256,000
  • Performance is measured with batch_size = 1,024, vocab_size = 128,000
