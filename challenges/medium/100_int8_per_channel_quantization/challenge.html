Implement per-channel symmetric INT8 quantization of a float32 weight matrix. This is the standard weight quantization scheme used in TensorRT, ONNX Runtime, and TFLite. Given a float32 matrix \(x\) of shape \(M \times K\), compute per-output-channel scales and INT8 quantized values.

Each column (channel) \(k\) gets an independent scale computed from the maximum absolute value within that column.

Algorithm

For each column k in [0, K):
  max_abs_k = max(|x[i,k]|) over all i in [0, M)
  scale_k = max_abs_k / 127.0
  If max_abs_k == 0: scale_k = 1.0
  For each i in [0, M):
    y[i,k] = clamp(round(x[i,k] / scale_k), -128, 127)

Implementation Requirements

  • External libraries are not permitted
  • The solve function signature must remain unchanged
  • INT8 results in y; per-channel float32 scales in scales
  • All computation must occur on the GPU
  • Channels (columns) are independent and may be computed in parallel

Example

Input: Matrix \(x\) (\(M=2, K=2\)): \[ \begin{bmatrix} 3.0 & -1.0 \\ 0.0 & 2.0 \end{bmatrix} \]

Column 0: max_abs=3.0, scale=0.02362
\(y[0,0] = \text{clamp}(\text{round}(3.0/0.02362), -128, 127) = 127\)
\(y[1,0] = \text{clamp}(\text{round}(0.0/0.02362), -128, 127) = 0\)

Column 1: max_abs=2.0, scale=0.01575
\(y[0,1] = \text{clamp}(\text{round}(-1.0/0.01575), -128, 127) = -63\)
\(y[1,1] = \text{clamp}(\text{round}(2.0/0.01575), -128, 127) = 127\)

Output: Matrix \(y\): \[ \begin{bmatrix} 127 & -63 \\ 0 & 127 \end{bmatrix} \]
scales = \([0.02362, 0.01575]\)

Constraints

  • 1 ≤ M ≤ 65,536
  • 1 ≤ K ≤ 65,536
  • Input x is float32; output y is int8; scales is float32
  • All tensors stored in row-major order
  • Performance is measured with M = 4,096, K = 4,096
