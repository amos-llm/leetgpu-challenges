Implement 2D block-wise FP8 E4M3 quantization of a float32 weight matrix. This is the 2D scaling scheme used in NVIDIA Transformer Engine and DeepGEMM for FP8 mixed-precision training, where the weight matrix is divided into BLOCK_SIZE × BLOCK_SIZE tiles and each tile gets its own scale (scale_granularity_mnk = (128, 128, K) in DeepGEMM).

Given a float32 matrix \(x\) of shape \(M \times N\), divide it into tiles of BLOCK_SIZE×BLOCK_SIZE. For each tile, compute the maximum absolute value, derive a tile scale, and encode each element to FP8 E4M3.

FP8 E4M3 Format

  • Bit layout: \([S|E_3|E_2|E_1|E_0|M_2|M_1|M_0]\), bias = 7, max normal = 448
  • Normal: \((-1)^S \times 2^{E-7} \times 1.M\)
  • Subnormal (exp=0000): \((-1)^S \times 2^{-6} \times 0.M\)
  • NaN: encode as 0x7F. Overflow: clamp to ±448

Algorithm

For each tile (bi, bj):
  max_abs = max(|x[i,j]|) for i, j in tile
  scale = max_abs / 448.0
  If max_abs == 0: scale = 1.0
  For each element (i, j) in tile:
    y[i,j] = fp8_e4m3(x[i,j] / scale)
  scales[bi, bj] = scale

Implementation Requirements

  • External libraries are not permitted
  • The solve function signature must remain unchanged
  • FP8 encodings in y (uint8); tile scales in scales (float32)
  • All computation must occur on the GPU
  • Edge tiles may be smaller than BLOCK_SIZE × BLOCK_SIZE

Example

Input: Matrix \(x\) (\(M=4, N=4\)), BLOCK_SIZE=2: \[ \begin{bmatrix} 240.0 & 120.0 & 1.0 & 2.0 \\ -240.0 & 0.0 & 3.0 & 4.0 \\ 0.0 & 60.0 & 5.0 & 6.0 \\ 30.0 & -30.0 & 7.0 & 8.0 \end{bmatrix} \]

Tile (0,0): max_abs=240.0, scale=0.5357...
Tile (0,1): max_abs=4.0, scale=0.008928...
Tile (1,0): max_abs=60.0, scale=0.1339...
Tile (1,1): max_abs=8.0, scale=0.017857...

Output: Tile scales: \[ \begin{bmatrix} 0.5357... & 0.008928... \\ 0.1339... & 0.017857... \end{bmatrix} \]
\(y\) with FP8 encodings for each tile

Constraints

  • 1 ≤ M, N ≤ 8,192
  • BLOCK_SIZE in {16, 32, 64, 128}
  • Input x is float32; output y is uint8; scales is float32
  • All tensors stored in row-major order
  • Performance is measured with M = 4,096, N = 4,096, BLOCK_SIZE = 128
